# YOLOv8 Vehicle Counter - คู่มือการติดตั้งและการใช้งาน

โปรแกรมนี้คือระบบนับจำนวนยานพาหนะ (Vehicle Counter) ที่พัฒนาด้วย Python โดยใช้งานร่วมกับโมเดลเอบีไอ (AI Model) ยอดนิยมอย่าง **YOLOv8** จาก Ultralytics เพื่อตรวจจับยานพาหนะประเภทต่างๆ (เช่น รถยนต์, รถบรรทุก, รถบัส, มอเตอร์ไซค์) และลากเส้นคัดกรองยานพาหนะที่วิ่งเข้า (Inbound) และวิ่งออก (Outbound)

---

## 💻 1. สิ่งที่ต้องมีเบื้องต้น (Prerequisites)

1. **Python 3.9 หรือสูงกว่า:**
   * ตรวจสอบได้โดยการเปิด Command Prompt / Terminal แล้วพิมพ์ `python --version`
   * หากยังไม่มี ให้ไปดาวน์โหลดและติดตั้งจากเว็บไซต์ [python.org](https://www.python.org/downloads/) 
   * **สำคัญ:** ตอนติดตั้ง Python อย่าลืมติ๊กช่อง `Add python.exe to PATH` 

2. **(ตัวเลือกเสริมสำหรับ GPU)** หากคอมพิวเตอร์ของคุณมีการ์ดจอ **NVIDIA** แนะนำให้ติดตั้ง **CUDA Toolkit** และ **cuDNN** เวอร์ชันที่เข้ามารองรับตัว PyTorch ตามลำดับ เพื่อให้การตรวจจับภาพ (Detection) ประมวลผลได้ลื่นไหลและมี FPS (Frame Per Second) ที่สูงกว่า CPU มาก

---

## 🛠️ 2. ขั้นตอนการติดตั้ง (Installation)

แนะนำให้สร้าง Virtual Environment (เพื่อไม่ให้ Environment ปะปนกับโปรเจกต์อื่นๆ) แต่หากรันโปรเจกต์เดียวก็สามารถข้ามขั้นตอนการสร้างไปได้:

### 2.1) สร้าง Virtual Environment (แนะนำ)
เปิด Command Prompt (CMD) หรือ PowerShell ในแฟ้ม (Folder) โปรเจกต์ แล้วพิมพ์คำสั่งด้านล่าง:
```bash
# สร้าง environment ชื่อ 'venv'
python -m venv venv

# เปิดใช้งาน environment 
# สำหรับ Windows:
venv\Scripts\activate
# สำหรับ Mac/Linux:
source venv/bin/activate
```
*(เมื่อเปิดสำเร็จจะมี `(venv)` ขึ้นนำหน้าบรรทัดคำสั่งของคุณ)*

### 2.2) ติดตั้งไลบรารีและแพ็กเกจที่จำเป็น
พิมพ์คำสั่งนี้ใน Terminal เพื่อติดตั้งแพ็กเกจทั้งหมดเข้าในเครื่อง (ใช้เวลานานเล็กน้อยโดยเฉพาะ PyTorch และ Ultralytics):

```bash
pip install ultralytics opencv-python numpy
```

- `ultralytics` คือแพ็กเกจหลักที่มี YOLOv8 อยู่ในตัว
- `opencv-python` คือไลบรารีเอาไว้เปิดวิดีโอ กล้อง และวาดรูป/เส้น ต่างๆ ลงบนภาพ 
- `numpy` คือไลบรารีการคำนวณที่สำคัญ โดย OpenCV เรียกใช้บ่อย ๆ

*(หมายเหตุ: Tkinter นั้นมักจะรวมมาพร้อมกับการติดตั้ง Python เบื้องต้นของ Windows / macOS อยู่แล้วจึงไม่ต้อง `pip install` เพิ่ม)*

## 🚀 3. วิธีการเริ่มใช้งาน (How to Run)

เมื่อติดตั้งทุกอย่างครบแล้ว สามารถรันโปรแกรมผ่าน Terminal ในโฟลเดอร์โปรเจกต์ได้เลย:

```bash
python vehcounter.py
```

### หน้าจอตั้งค่า (Setup Interface) 
หลังจากรันโปรแกรม จะมีหน้าต่างขึ้นมาให้ผู้ใช้งานเลือก 2 โหมด:

1. **ไฟล์วิดีโอ (Video File):** ให้กดปุ่ม `Browse` ไปหาไฟล์นามสกุล .mp4 / .avi  
2. **สตรีมมิ่ง / กล้อง (Stream/Webcam):** 
   * สำหรับกล้อง Webcam ที่ต่อในเครื่อง ให้พิมพ์ `0` (หรือ `1` ตามลำดับกล้อง) 
   * สำหรับสตรีมผ่านเน็ต ให้พิมพ์ Link M3U8 หรือ RTSP ลงในช่องนี้ (ตัวอย่างเช่น IP Camera)

**โหมดความเร็วสูงสุด (Fast Mode):** 
เป็นการปรับความละเอียดของภาพที่ตัว AI นำไปประมวลผลให้เหลือ `320px` ซึ่งจะสแกนภาพได้เร็วขึ้น เหมาะสำหรับใช้งานกับ CPU แต่ถ้าพบว่าภาพเบลอจนนับไม่แม่น ให้ **เอาเครื่องหมายถูกออก** (มันจะปรับความละเอียดเป็น `640px` แทน) ภาพจะชัดขึ้น แต่วิดีโออาจจะมีอาการกระตุกรันช้าลง

เมื่อตั้งค่าเสร็จสิ้นแล้ว กด **"🚀 เริ่มนับยานพาหนะ (Start)"**

## ✍️ 4. ขั้นตอนการวาดเส้นนับรถ (Drawing Counting Line)

เมื่อเริ่มต้นโปรแกรม หน้าจอ **Setup Line** ของวิดีโอจะเด้งขึ้นมา:

1. **คลิกซ้ายจุดแรก** บนหน้าจอในตำแหน่งที่ต้องการ 
2. **คลิกซ้ายจุดที่สอง** ในอีกฝั่งถนนเพื่อลากเป็นเส้นตรงขวางเส้นทางจราจร
   * *หากคลิกผิด ให้คลิกจุดที่ 3 มันจะรีเซ็ตเริ่มวาดเส้นใหม่ตั้งแต่แรก*
3. เมื่อได้เส้นที่พอใจแล้ว **ให้กดปุ่มแป้นพิมพ์ `c`** (Confirm) เพื่อยืนยันเส้นนั้น

## 📊 5. หน้าต่างแสดงผลและการทำงาน (Dashboard & Counting)

โปรแกรมจะแบ่งหน้าต่างเป็น 2 ส่วน เมื่อกดยืนยันเส้น:
1. **Vehicle Counting:** หน้าต่างแสดงผลลัพธ์วิดีโอ พร้อมตีกรอบ (Bounding Box) สีเขียวรอบรถที่ตรวจจับเจอ พร้อมลากเส้นสีฟ้าตามรอยจุดไข่ปลา
2. **Dashboard:** หน้าต่างแสดงตัวเลขสรุปสถิติสดๆ (Real-time) แบ่งเป็นรถประเภทต่างๆ ในฝั่ง Inbound (วิ่งเข้าเส้น) และ Outbound (วิ่งออกจากเส้น)

### 🔴 การหยุดและการสตรีมค้าง
หน้าต่าง **Dashboard** จะมีปุ่มใหญ่ๆ 2 ปุ่ม:

- **RELOAD STREAM (ปุ่มส้ม) หรือ กดคีย์บอร์ด `r`:** เอาไว้ใช้ในกรณีที่ภาพสตรีมกระตุก ภาพค้าง เมื่อกดปุ่มนี้ระบบจะเชื่อมต่อภาพสตรีมที่ URL เดิมนั้นใหม่แบบทันที เพื่อให้ไม่ต้องเปิด-ปิดโปรแกรมใหม่ 
*(ระบบยังมี **Auto-Reload** ในตัว หากภาพไม่ขยับนานเกิน 5 วินาที ระบบจะดึงสตรีมใหม่โดยอัตโนมัติเช่นกัน)*

- **STOP & EXPORT (ปุ่มแดง) หรือ กดคีย์บอร์ด `q` / `ESC`:** เอาไว้ใช้สำหรับ **บันทึกข้อมูล** และปิดโปรแกรม ระบบจะสรุปจำนวนและ ID รถลงไฟล์ CSV ในชื่อที่คุณรับตั้งค่าทางหน้า GUI แรกแล้วหน้าต่างทั้งหมดจะปิดตัวเอง

## 📂 6. ดูรายงาน (Exported Data)

ถ้าคุณได้กรอกชื่อไฟล์ เช่น `vehicle_counts.csv`
ระบบจะรวบรวมเหตุการณ์ในไฟล์นี้ในรู้แบบแถวตอน ซึ่งจะประกอบด้วย:
- **Timestamp:** วันและเวลาที่นับได้
- **Vehicle_ID:** รหัสนับเลขประจำตัวของรถสิบล้อหรือรถยนต์คันนั้น (ID Tracking)
- **Class:** ประเภทของรถ (เช่น car, bus, truck, motorcycle)
- **Direction:** อินบาวด์ หรือ เอาท์บาวด์ (Inbound / Outbound)

คุณสามารถนำไฟล์ `.csv` นี้ไปเปิดใน Microsoft Excel หรือระบบรายงานต่างๆ นำไปทำ Pivot หรือ Dashboard บนแพลตฟอร์มอื่นต่อได้

## 🛠 7. ปัญหาและวิธีแก้ปัญหาทั่วไป (Troubleshooting)

- **มีปัญหาเกี่ยวกับไฟล์โมเดลไม่เจอ:** ตอนรันครั้งแรก YOLOv8 จะไปโหลดโมเดล `yolov8n.pt` จากอินเทอร์เน็ตมาเก็บไว้ในเครื่องอัตโนมัติ (อาจจะใช้เวลาโหลดนิดหน่อยในครั้งแรก) ดังนั้นให้แน่ใจว่าเชื่อมต่ออินเทอร์เน็ตปกติ
- **สตรีมมิ่งไม่ติด / Error m3u8 ไม่เจอเฟรมแรก:** หมายถึงลิงก์สตรีมมิ่งเสื่อม, หลุด หรือใช้โทเคนหมดอายุ คุณสามารถหาลิงก์สตรีมมิ่งสดใหม่ๆ มาเข้าโปรแกรมแทน
- **จอภาพดำ / OpenCV แสดงผลผิดเพี้ยน:** ตอนใช้คลิกเมาส์ตั้งเส้น บางเครื่องที่มีการสเกลหน้าจอ Window ใหญ่ๆ ไว้ หากภาพใหญ่ทะลุจอไม่สามารถกด Confirm ได้ ให้ลองลากหน้าต่างโปรแกรม OpenCV เพื่อไปกดตัวอักษร "c" บนคีย์บอร์ด
