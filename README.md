# YOLOv8 Vehicle Counter — คู่มือการติดตั้งและการใช้งาน

โปรแกรมนับจำนวนยานพาหนะแบบ Real-time พัฒนาด้วย Python + **YOLOv8** จาก Ultralytics รองรับการวางเส้นนับได้สูงสุด **3 เส้น** (เช่น ช่องตรง / เลี้ยวซ้าย / เลี้ยวขวา) และ export ผลเป็นไฟล์ **Excel (.xlsx)** แยก Sheet ต่อเส้นพร้อม Sheet สรุปรวม

---

## 💻 1. สิ่งที่ต้องมีเบื้องต้น (Prerequisites)

1. **Python 3.9 หรือสูงกว่า**
   - ตรวจสอบด้วย `python --version`
   - ดาวน์โหลดได้ที่ [python.org](https://www.python.org/downloads/)
   - **สำคัญ:** ตอนติดตั้งให้ติ๊ก `Add python.exe to PATH` ด้วย

2. **(ตัวเลือกเสริม — GPU)** หากมีการ์ดจอ **NVIDIA** ติดตั้ง CUDA Toolkit และ cuDNN เพิ่มเพื่อให้ AI ประมวลผลเร็วขึ้นมากเมื่อเทียบกับ CPU

---

## 🛠️ 2. การติดตั้ง (Installation)

### 2.1) สร้าง Virtual Environment (แนะนำ)

```bash
# สร้าง environment
python -m venv venv

# เปิดใช้งาน — Windows
venv\Scripts\activate

# เปิดใช้งาน — Mac / Linux
source venv/bin/activate
```

### 2.2) ติดตั้งแพ็กเกจที่จำเป็น

```bash
pip install ultralytics opencv-python numpy openpyxl
```

| แพ็กเกจ | หน้าที่ |
|---|---|
| `ultralytics` | โมเดล YOLOv8 สำหรับตรวจจับยานพาหนะ |
| `opencv-python` | เปิดวิดีโอ / กล้อง และวาดภาพ |
| `numpy` | การคำนวณเมทริกซ์ |
| `openpyxl` | สร้างและเขียนไฟล์ Excel (.xlsx) |

> **หมายเหตุ:** Tkinter มาพร้อม Python บน Windows / macOS ไม่ต้องติดตั้งเพิ่ม

## ⚙️ 4. หน้าตั้งค่าโปรแกรม (Setup GUI)

### 4.1) เลือกแหล่งวิดีโอ (Video Source)

| ตัวเลือก | วิธีใช้ |
|---|---|
| **อัปโหลดไฟล์วิดีโอ** | กด `Browse…` แล้วเลือกไฟล์ `.mp4 / .avi / .mov / .mkv` |
| **สตรีม / กล้องวงจรปิด** | พิมพ์ `0` สำหรับ Webcam, หรือ URL RTSP / M3U8 สำหรับ IP Camera |

### 4.2) จำนวนเส้นนับ (Counting Lines)

เลือกจำนวนเส้นนับจาก Dropdown **1 / 2 / 3 เส้น** และตั้งชื่อแต่ละเส้นในช่องที่ปรากฏขึ้น เช่น:

```
● Line 1:  ตรง
● Line 2:  เลี้ยวซ้าย
● Line 3:  เลี้ยวขวา
```

> ชื่อเส้นจะปรากฏบนวิดีโอและใช้เป็นชื่อ Sheet ใน Excel ที่  export ออกมา

### 4.3) โหมดความเร็ว (Fast Mode)

| โหมด | ความละเอียด AI | เหมาะกับ |
|---|---|---|
| ✅ Fast Mode (ค่าเริ่มต้น) | 320 px | CPU / ประสิทธิภาพ FPS สูง |
| ☐ ปิด Fast Mode | 640 px | GPU / ต้องการความแม่นยำสูง |

### 4.4) ชื่อไฟล์ Export

ระบุชื่อไฟล์ `.xlsx` ในช่อง **Export Excel** เช่น `สี่แยก_A.xlsx`  
(ค่าเริ่มต้น: `vehicle_counts.xlsx`)

กด **🚀 เริ่มนับยานพาหนะ (Start)** เพื่อดำเนินการต่อ

## ✍️ 5. การวาดเส้นนับ (Drawing Counting Lines)

หลังกด Start หน้าต่าง **Setup Line** จะเปิดขึ้น ให้วาดเส้นทีละเส้น:

1. **คลิกซ้ายจุดแรก** บนถนน
2. **คลิกซ้ายจุดที่สอง** ฝั่งตรงข้าม — เส้นจะลากขวางถนน
3. กด **`C`** เพื่อยืนยันเส้น แล้วดำเนินการวาดเส้นถัดไป (ถ้ามี)
4. กด **`Q`** เพื่อออกจากโปรแกรม

> **เคล็ดลับ:** ถ้าคลิกผิดตำแหน่ง ให้คลิกครั้งที่ 3 — ระบบจะรีเซ็ตเส้นปัจจุบันและเริ่มใหม่
>
> เส้นที่วาด confirm แล้วก่อนหน้าจะยังแสดงบนหน้าจอขณะวาดเส้นถัดไป เพื่อใช้เป็นอ้างอิง

## 📊 6. หน้าต่างแสดงผล (Dashboard & Counting)

โปรแกรมแบ่งเป็น 2 หน้าต่าง:

### Vehicle Counting
- แสดงวิดีโอสด พร้อม Bounding Box รอบยานพาหนะแต่ละคัน
- เส้นนับแต่ละเส้นมีสีต่างกัน (Cyan / Green / Orange)
- มุมซ้ายบนแสดงยอด `IN / OUT / TOT` รวมทุกเส้นแบบ Real-time

### Dashboard
- แสดงยอดนับแยกรายเส้น พร้อมชื่อเส้นและยอด Inbound/Outbound ของแต่ละประเภทรถ
- แสดง Grand Total รวมทุกเส้น
- แสดงเวลาปัจจุบันและโหมด (Fast / Quality)

### ปุ่มควบคุม

| ปุ่ม | คีย์บอร์ด | หน้าที่ |
|---|---|---|
| **RELOAD STREAM** | `R` | เชื่อมต่อสตรีมใหม่ — ใช้เมื่อภาพค้างหรือกระตุก |
| **STOP & EXPORT** | `Q` หรือ `ESC` | หยุดนับและบันทึกข้อมูลเป็น Excel |

> **Auto-Reload:** ถ้าภาพสตรีมไม่ขยับนานเกิน 5 วินาที ระบบจะดึงสตรีมใหม่อัตโนมัติ

## 📂 7. ไฟล์ผลลัพธ์ (Exported Excel)

เมื่อกด STOP & EXPORT ระบบจะสร้างไฟล์ `.xlsx` ดังนี้:

### Sheet: Summary
| Line | Label | Direction | Class | Count |
|---|---|---|---|---|
| Line 1 | ตรง | Inbound | car | 42 |
| Line 1 | ตรง | Outbound | truck | 7 |
| Line 2 | เลี้ยวซ้าย | Inbound | motorcycle | 15 |

### Sheet: Line1\_ตรง (และ Line2, Line3 ตามจำนวนเส้นที่ตั้งค่า)
| Timestamp | Vehicle\_ID | Class | Direction |
|---|---|---|---|
| 2026-03-17 10:30:05 | 12 | car | Inbound |
| 2026-03-17 10:30:08 | 15 | motorcycle | Outbound |

> จำนวน Sheet ทั้งหมด = **1 Summary Sheet** + **N Detail Sheets** (N = จำนวนเส้นที่เลือก)

## 🛠 8. ปัญหาและวิธีแก้ (Troubleshooting)

| ปัญหา | วิธีแก้ |
|---|---|
| โมเดลไม่เจอ / โหลดช้าครั้งแรก | YOLOv8 ดาวน์โหลด `yolov8n.pt` อัตโนมัติ — ต้องมีอินเทอร์เน็ต |
| สตรีมไม่ติด / Error M3U8 | ลิงก์หมดอายุ — ให้หาลิงก์ใหม่มาใส่ |
| ภาพดำ / หน้าต่างใหญ่ทะลุจอ | ลากขยับหน้าต่าง OpenCV แล้วกด `C` หรือ `Q` บนคีย์บอร์ด |
| `ModuleNotFoundError: openpyxl` | รัน `pip install openpyxl` ใน Terminal |
| ความแม่นยำต่ำ / นับพลาด | ปิด Fast Mode เพื่อใช้ความละเอียด 640px |
| รถผ่านแต่ไม่ถูกนับ | ตรวจสอบว่าเส้นขวางเส้นทางเดินรถจริงๆ และ trajectory ของรถตัดเส้น |
