# YOLOv8 Vehicle Counter - คู่มือการติดตั้งและการใช้งาน

โปรแกรมนี้คือระบบนับจำนวนยานพาหนะ (Vehicle Counter) ที่พัฒนาด้วย Python โดยใช้งานร่วมกับโมเดลเอบีไอ (AI Model) ยอดนิยมอย่าง **YOLOv8** จาก Ultralytics เพื่อตรวจจับยานพาหนะประเภทต่างๆ (เช่น รถยนต์, รถบรรทุก, รถบัส, มอเตอร์ไซค์) และลากเส้นคัดกรองยานพาหนะที่วิ่งเข้า (Inbound) และวิ่งออก (Outbound)

---

## 💻 1. สิ่งที่ต้องมีเบื้องต้น (Prerequisites)

1. **Python 3.9 หรือสูงกว่า:**
   * ตรวจสอบได้โดยการเปิด Command Prompt / Terminal แล้วพิมพ์ `python --version`
   * หากยังไม่มี ให้ไปดาวน์โหลดและติดตั้งจากเว็บไซต์ [python.org](https://www.python.org/downloads/)
   * **สำคัญ:** ตอนติดตั้ง Python อย่าลืมติ๊กช่อง `Add python.exe to PATH`

2. **(ตัวเลือกเสริมสำหรับ GPU)** หากคอมพิวเตอร์ของคุณมีการ์ดจอ **NVIDIA** แนะนำให้ติดตั้ง **CUDA Toolkit** และ **cuDNN** เวอร์ชันที่เข้ามารองรับตัว PyTorch ตามลำดับ เพื่อให้การตรวจจับภาพ (Detection) ประมวลผลได้ลื่นไหลและมี FPS (Frame Per Second) ที่สูงกว่า CPU มาก

---

## 🛠️ 2. ขั้นตอนการติดตั้ง (Installation)

แนะนำให้สร้าง Virtual Environment (เพื่อไม่ให้ Environment ปะปนกับโปรเจกต์อื่นๆ) แต่หากรันโปรเจกต์เดียวก็สามารถข้ามขั้นตอนการสร้างไปได้:

### 2.1) สร้าง Virtual Environment (แนะนำ)
เปิด Command Prompt (CMD) หรือ PowerShell ในแฟ้ม (Folder) โปรเจกต์ แล้วพิมพ์คำสั่งด้านล่าง:
```bash
# สร้าง environment ชื่อ 'venv'
python -m venv venv

# เปิดใช้งาน environment 
# สำหรับ Windows:
venv\Scripts\activate
# สำหรับ Mac/Linux:
source venv/bin/activate
```
*(เมื่อเปิดสำเร็จจะมี `(venv)` ขึ้นนำหน้าบรรทัดคำสั่งของคุณ)*

### 2.2) ติดตั้งไลบรารีและแพ็กเกจที่จำเป็น
พิมพ์คำสั่งนี้ใน Terminal เพื่อติดตั้งแพ็กเกจทั้งหมดเข้าในเครื่อง (ใช้เวลานานเล็กน้อยโดยเฉพาะ PyTorch และ Ultralytics):

```bash
pip install ultralytics opencv-python numpy
```

- `ultralytics` คือแพ็กเกจหลักที่มี YOLOv8 อยู่ในตัว
- `opencv-python` คือไลบรารีเอาไว้เปิดวิดีโอ กล้อง และวาดรูป/เส้น ต่างๆ ลงบนภาพ 
- `numpy` คือไลบรารีการคำนวณที่สำคัญ โดย OpenCV เรียกใช้บ่อย ๆ

*(หมายเหตุ: Tkinter นั้นมักจะรวมมาพร้อมกับการติดตั้ง Python เบื้องต้นของ Windows / macOS อยู่แล้วจึงไม่ต้อง `pip install` เพิ่ม)*

## 🚀 3. วิธีการเริ่มใช้งาน (How to Run)

เมื่อติดตั้งทุกอย่างครบแล้ว สามารถรันโปรแกรมผ่าน Terminal ในโฟลเดอร์โปรเจกต์ได้เลย:

```bash
python vehcounter.py
```

### หน้าจอตั้งค่า (Setup Interface) 
หลังจากรันโปรแกรม จะมีหน้าต่างขึ้นมาให้ผู้ใช้งานเลือกตั้งค่า:

1. **Video Source:** 
   * **อัปโหลดไฟล์วิดีโอ:** ให้กดปุ่ม `Browse` ไปหาไฟล์นามสกุล .mp4 / .avi  
   * **ลิงก์สตรีม / กล้องวงจรปิด:** สำหรับกล้อง Webcam พิมพ์ `0` หรือสำหรับสตรีมพิมพ์ Link M3U8/RTSP 
2. **Counting Lines (จำนวนเส้นนับ):** สามารถเลือกได้ตั้งแต่ 1 ถึง 3 เส้น พร้อมตั้งชื่อให้แต่ละเส้นได้อิสระ
3. **Advanced Settings (จูนการจับมอเตอร์ไซค์):**
   * **ความแม่นยำ AI (Model):** เลือกขนาดโมเดล YOLOv8 (`yolov8n` - เร็วสุด, `yolov8s` - สมดุลและจับรถเล็กได้ดี, `yolov8m` - แม่นยำสูงสุดแต่กินเครื่อง)
   * **ความไว (Sensitivity):** ปรับความไวในการตรวจจับ (High, Normal, Low)
4. **Fast Mode:** ปรับลดสเกลภาพให้ประมวลผลเร็วขึ้น (ลื่นไหลขึ้น แต่อาจเลิกจับรถคันเล็กๆ ที่อยู่ไกล) แนะนำให้ **เอาเครื่องหมายถูกออก** หากเน้นจับมอเตอร์ไซค์
5. **Export Excel:** กำหนดชื่อไฟล์ Excel ที่ต้องการบันทึกผล (เช่น `vehicle_counts.xlsx`)

เมื่อตั้งค่าเสร็จสิ้นแล้ว กด **"🚀 เริ่มนับยานพาหนะ (Start)"**

---

## ✍️ 4. ขั้นตอนการวาดเส้นนับรถ (Drawing Counting Line)

เมื่อเริ่มต้นโปรแกรม หน้าจอ **Setup Line** จะเด้งขึ้นมา:

1. โปรแกรมจะให้คุณวาดเส้นตามจำนวนที่คุณตั้งไว้ (1-3 เส้น) ทีละเส้น
2. สำหรับแต่ละเส้น:
   * **คลิกซ้ายจุดแรก** บนหน้าจอในตำแหน่งที่ต้องการ 
   * **คลิกซ้ายจุดที่สอง** ในอีกฝั่งถนนเพื่อลากเป็นเส้นตรงขวางเส้นทางจราจร
   * *หากคลิกผิด ให้คลิกจุดที่ 3 มันจะรีเซ็ตเริ่มวาดเส้นนั้นใหม่*
   * เมื่อได้เส้นที่พอใจแล้ว **ให้กดปุ่มแป้นพิมพ์ `c`** (Confirm) เพื่อยืนยันเส้นนั้น และไปวาดเส้นถัดไป (ถ้ามี)

---

## 📊 5. หน้าต่างแสดงผลและการทำงาน (Dashboard & Counting)

โปรแกรมจะแบ่งหน้าต่างเป็น 2 ส่วน:
1. **หน้าต่างวิดีโอหลัก:** แสดงผลลัพธ์วิดีโอ พร้อมตีกรอบ (Bounding Box) และเส้น Tracking ของรถ พร้อมตัวเลขสรุปรวมมุมซ้ายบน
2. **Dashboard (สถิติ):** หน้าต่างขวามือแสดงตัวเลขสรุปสถิติสดๆ (Real-time) แบ่งเป็นรถประเภทต่างๆ ในฝั่ง In (วิ่งเข้าเส้น) และ Out (วิ่งออกจากเส้น) แยกตามแต่ละเส้นที่วาดไว้

### 🔴 การหยุดและการสตรีมค้าง
ในหน้าต่าง **Dashboard** จะมีปุ่ม:

- **RELOAD STREAM (หรือ กดคีย์บอร์ด `r`):** รีโหลดภาพสตรีมใหม่ทันทีในกรณีที่ภาพค้าง *(ระบบมี Auto-Reload หากภาพไม่ขยับนานเกิน 5 วินาที)*
- **STOP & EXPORT (หรือ กดคีย์บอร์ด `q` / `ESC`):** สั่งหยุดโปรแกรมและทำการ **Export ข้อมูลลงไฟล์ Excel** ที่ตั้งชื่อไว้ทันที

---

## 📂 6. ดูรายงาน (Exported Data)

เมื่อกด STOP & EXPORT หรือไฟล์วิดีโอเล่นจบ ระบบจะสร้างไฟล์ **Excel (.xlsx)** ตามชื่อที่คุณตั้งไว้ โดยภายในไฟล์จะประกอบด้วย:

1. **Sheet "Summary" (หน้าสรุป):** 
   * สรุปจำนวนรถแต่ละประเภท (Car, Truck, Bus, Motorcycle) ที่วิ่งเข้า (Inbound) และออก (Outbound) สำหรับ **แต่ละเส้น** รวมไปถึงผลรวมทั้งหมด
2. **Sheet รายละเอียดของแต่ละเส้น (เช่น "Line 1 Detail"):**
   * **Timestamp:** วันและเวลาที่นับได้
   * **Vehicle ID:** หมายเลข Tracking ประจำตัวของรถ
   * **Class:** ประเภทของรถ
   * **Direction:** Inbound หรือ Outbound

คุณสามารถนำไฟล์ `.xlsx` นี้ไปใช้งานต่อ ทำกราฟ หรือ Pivot Table ได้ทันที

## 🛠 7. ปัญหาและวิธีแก้ปัญหาทั่วไป (Troubleshooting)

- **ตรวจจับมอเตอร์ไซค์ไม่ค่อยเจอ:** ให้ลองเปลี่ยนตั้งค่าเป็นโมเดล `yolov8s.pt`, ปรับ Sensitivity เป็น `High` และ **เอาเครื่องหมายถูกหน้า Fast Mode ออก** 
- **มีปัญหาเกี่ยวกับไฟล์โมเดลไม่เจอ:** ตอนรันครั้งแรก YOLOv8 จะไปโหลดโมเดลจากอินเทอร์เน็ตมาเก็บไว้ในเครื่องอัตโนมัติ ให้แน่ใจว่าเชื่อมต่ออินเทอร์เน็ตอยู่
- **สตรีมมิ่งไม่ติด / Error m3u8 ไม่เจอเฟรมแรก:** หมายถึงลิงก์สตรีมมิ่งเสื่อม, หลุด หรือใช้โทเคนหมดอายุ
- **จอภาพดำ / OpenCV แสดงผลผิดเพี้ยน หรือหน้าต่างใหญ่ทะลุจอ:** บางเครื่องมีการสเกลหน้าจอ Window หากภาพใหญ่ทะลุจอจนหาหน้าต่างไม่แจนตอนวาดเส้น ไม่ต้องตกใจ ให้คลิกที่หน้าต่างไหนก็ได้ของโปรแกรมแล้วกด `c` เพื่อยืนยันเส้นไปก่อน โปรแกรมจะย่อขนาดให้อัตโนมัติในตอนนับรถ
